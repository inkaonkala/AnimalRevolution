# 🐾 Animal Revolution

Animal Revolution is a cozy pixel-art management and exploration game built with **Godot**.

You play as a dog leading a secret animal society that has taken over an apartment building. Rescue new animal friends, grow crops on the rooftop, factor humans by night, take care of your community, and slowly build a strange but thriving revolution.

---

## ✨ Current Features

- 🐶 Explore a multi-floor apartment building
- 🐹 Recruit rodents to help on the rooftop
- 🌱 Plant, water, and harvest crops
- 📅 Day and night cycle
- 🎒 Inventory system
- 🛗 Elevator travel between floors
- 😺 NPC interactions and evolving animal behaviors

More features are being added as development continues.

---


## 🎮 Controls

| Key | Action |
|------|--------|
| WASD / Arrow Keys | Move |
| E | Interact |
| Space | Use tool / Attack |
